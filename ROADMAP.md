# OpenProof — Roadmap

> A minimal, verifiable proof-of-existence app on Base Sepolia.

## Current — MVP (June 2026)

Core attestation flow on testnet.

- [x] Smart contract: OpenProofRegistry on Base Sepolia
- [x] Receipt schema v2 (versioned, validated, backward-compatible)
- [x] Web app: claim, verify, and browse attestations
- [x] Wallet connection (Coinbase Smart Wallet, MetaMask)
- [x] Receipt card and feed components
- [ ] Base Name resolution
- [ ] Prefill attestation form from URL params
- [ ] Base MCP integration for agentic attestations
- [x] CI/CD pipeline

## Next — v0.2

- [ ] Receipt detail page with full metadata
- [ ] Attestation search and filter
- [ ] Batch attestation support
- [ ] Production deployment planning
- [ ] Gas cost optimization

## Future

- [ ] Multi-chain support (Base mainnet, others)
- [ ] Attestation revocation
- [ ] Off-chain receipt generation
- [ ] API for programmatic attestations
- [ ] Verification API
